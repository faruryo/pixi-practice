# 作り方

## プロジェクト準備

参考：[Vue.js を vue-cli を使ってシンプルにはじめてみる](https://qiita.com/567000/items/dde495d6a8ad1c25fa43)

### vue-cliのインストール

```bash
npm install -g @vue/cli
```

### プロジェクトを作成する

```bash
vue create pixi-practice
> default

cd pixi-practice
npm run serve
```

ブラウザで http://localhost:8080/ をアクセスするとサンプルが表示されるはずなので確認しておく。
